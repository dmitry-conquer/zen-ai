ZEN Express

Vite + Tailwind CSS v4 + Alpine.js + TypeScript + Handlebars → WordPress (ACF Flexible Content)

Getting started

  1. Fill in content/homepage.md with real client data
  2. Add images to public/images/
  3. Add design references to design-examples/
  4. Use prompt from prompt.md → ## 1. Static layout to build the static site
  5. Run npm run build — compiles Tailwind and copies assets to wp-theme/assets/
  6. Use prompt from prompt.md → ## 2. ACF / WordPress integration to convert to WP theme
  7. Use prompt from prompt.md → ## 3. Content import to populate WordPress

NPM scripts

  • npm run dev — local dev server
  • npm run host — dev server exposed on LAN
  • npm run build — TypeScript check + Vite build + copy assets to wp-theme/
  • npm run preview — preview the production build

Project structure

  • pages/ — HTML entry points (one file per page)
  • components/ — Handlebars partials (header, footer, sections)
  • content/ — copy files per page
  • public/images/ — all images
  • design-examples/ — reference screenshots for AI color/style analysis
  • src/scripts/main.ts — JS entry (Alpine.js, Lenis, AOS)
  • src/styles/main.css — CSS entry (Tailwind v4)
  • wp-theme/ — WordPress theme (receives compiled assets on build)
